# Bora Uner — Portfolio

A single-page developer portfolio built to the **monopo saigon** editorial
design language (see `DESIGN.md`). Monochrome UI, one iridescent WebGL hero
backdrop, patient Motion-driven animation.

## Stack
- Vite + React
- Tailwind v4
- [Motion](https://motion.dev) for animation, [Lenis](https://lenis.darkroom.engineering/) for smooth scroll
- [Three.js](https://threejs.org) for the hero's WebGL iridescent scene

See `CLAUDE.md` for the full project brief and non-negotiable design rules.

## Develop
```
npm install
npm run dev      # local dev server
npm run build    # production build
npm run lint     # oxlint
```
